Scalaz 3.0 API Specification

This document is the API specification for Scalaz 3.0

Class Summary
trait Applicative [AP] extends Functor[AP] with Apply[AP] with Unit[AP] with AnyRef
Defines an applicative functor as described by McBride and Paterson in Applicative Programming with Effects.
sealed trait ApplicativeW [AP, A] extends AnyRef
Wraps Applicative and a value for which there exists an instance and provides additional methods.
trait Apply [AP] extends AnyRef
Function application within an environment.
trait Bind [B] extends AnyRef
Binds a function through an environment (known also as sequencing).
trait Cofunctor [F] extends AnyRef
Contra-variant function application in an environment.
sealed trait CofunctorW [F, A] extends AnyRef
Wraps Cofunctor and a value for which there exists an instance and provides additional methods.
trait Each [E] extends AnyRef
Executes a side-effect through an environment.
sealed trait EachW [E, A] extends AnyRef
Wraps Each and a value for which there exists an instance and provides additional methods.
trait Empty [E] extends AnyRef
An empty environment.
trait FoldLeft [F] extends AnyRef
Folding a function across an iterable environment from left to right.
sealed trait FoldLeftW [F, A] extends AnyRef
Wraps FoldLeft and a value for which there exists an instance and provides additional methods.
trait FoldRight [F] extends AnyRef
Folding a function across an iterable environment from right to left.
sealed trait FoldRightW [F, A] extends AnyRef
Wraps FoldRight and a value for which there exists an instance and provides additional methods.
trait Functor [F] extends AnyRef
Covariant function application in an environment. i.e. a covariant Functor.
sealed trait FunctorW [F, A] extends AnyRef
Wraps Functor and a value for which there exists an instance and provides additional methods.
trait Monad [M] extends Applicative[M] with Bind[M]
Abstract a model that sequences computation through an environment.
trait MonadEmpty [M] extends Monad[M] with Empty[M]
A monad that also has an empty value.
trait MonadEmptyPlus [M] extends MonadPlus[M] with MonadEmpty[M]
A monad with empty and plus.
sealed trait MonadEmptyPlusW [M, A] extends AnyRef
Wraps MonadEmptyPlus and a value for which there exists an instance and provides additional methods.
trait MonadPlus [M] extends Monad[M] with Plus[M]
A monad with plus.
sealed trait MonadPlusW [M, A] extends AnyRef
Wraps MonadPlus and a value for which there exists an instance and provides additional methods.
sealed trait MonadW [M, A] extends AnyRef
Wraps Monad and a value for which there exists an instance and provides additional methods.
trait Monoid [A] extends Semigroup[A] with Zero[A]
A categorical monoid.
trait Paramorphism [P] extends AnyRef
A paramorphism of an iterable that can be deconstructed into next element and remaining elements.
sealed trait ParamorphismW [P, A] extends AnyRef
Wraps Paramorphism and a value for which there exists an instance and provides additional methods.
trait Plus [P] extends AnyRef
Appends through an environment.
trait Semigroup [A] extends AnyRef
Appends two elements.
sealed trait SemigroupW [A] extends AnyRef
Wraps Semigroup and a value for which there exists an instance and provides additional methods.
sealed trait State [S, A] extends AnyRef
Stateful computation.
sealed trait Traversable [T] extends Functor[T] with AnyRef
Defines a traversable type as described by McBride and Paterson in Applicative Programming with Effects. Also see The Essence of the Iterator Pattern by Saad Alfoudari and L.J. Steggles for further work.
sealed trait TraversableW [T, A] extends AnyRef
Wraps Traversable and a value for which there exists an instance and provides additional methods.
trait Unit [U] extends AnyRef
Projects a value into an environment.
trait Zero [Z] extends AnyRef
The zero/identity element for a set.
