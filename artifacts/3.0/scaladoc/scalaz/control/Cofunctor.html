trait Cofunctor[F]
extends AnyRef
Contra-variant function application in an environment.

All contra-variant functor instances must satisfy 2 laws:

  1. identity
    forall a. a == comap(identity, a)
  2. composition
    forall a f g. comap(f compose g, a) == comap(g, comap(f, a))
